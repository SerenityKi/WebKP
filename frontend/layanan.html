Layanan Helpdesk TIK

Apa itu Helpdesk TIK?

Helpdesk TIK adalah pelayanan Bidang Teknologi Informasi dan Komunikasi (TIK) pada Dinas Komunikasi Informatika Persandian dan Statistik Kabupaten Bekasi.
Helpdesk TIK bertujuan untuk meningkatkan layanan teknologi informasi pada seluruh Perangkat Daerah di Pemerintah Kabupaten Bekasi.

Layanan Jaringan Intra Pemerintah Daerah

Menyediakan jaringan LAN dan Wifi yang handal untuk mendukung kegiatan operasional pemerintah daerah.

  • Pemasangan Jaringan LAN dan Wifi
  • Pengaduan Kendala Jaringan

Layanan Aplikasi

Mengembangkan dan memelihara aplikasi yang mendukung efisiensi dan transparansi di lingkungan pemerintah daerah.

  • Pengembangan Aplikasi
  • Pengaduan Kendala Server

Layanan Data Center

Menyediakan layanan penyimpanan dan pengelolaan data yang aman dan andal.

  • Co-Location Server
  • Pengaduan Kendala Server

Layanan Virtual Machine/Virtual Private Server

Memberikan solusi hosting virtual untuk mendukung kebutuhan server pemerintah daerah.

  • Pembuatan Virtual Machine/VPS
  • Pengaduan Kendala VM/VPS

Layanan Subdomain

Menyediakan subdomain untuk memudahkan akses ke berbagai layanan pemerintah Kabupaten Bekasi.

  • Subdomain Bekasikab.go.id

Layanan e-mail/Cloud

Menyediakan layanan email dan cloud storage untuk mendukung komunikasi dan penyimpanan data.

  • Pembuatan e-mail dan Cloud
  • Pengaduan Kendala e-mail dan Cloud
